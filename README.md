# 🚗 DiecastBricksTR

> Türkiye'nin diecast koleksiyoncuları için alım/satım platformu.

**🌐 [diecastbrickstr.netlify.app](https://diecastbrickstr.netlify.app)**  
**👥 [r/DiecastBricksTR](https://www.reddit.com/r/DiecastBricksTR/)**

---

## Nedir?

Hot Wheels, Matchbox, Bburago, AutoArt ve daha fazlası — Türkiye'deki diecast koleksiyoncularının araçlarını alıp satabileceği, topluluk odaklı bir platform.

Sahibinden.com mantığında ama tamamen diecast'e özel.

## Özellikler

- 🔍 Marka, ölçek, fiyat, şehir, durum filtrelemeleri
- 💬 Kullanıcılar arası mesajlaşma
- 📋 İlan verme sistemi
- ❤️ Favorilere ekleme
- 📱 Mobil uyumlu tasarım

## Teknoloji

Şu an tamamen frontend — HTML, CSS, JavaScript.  
Yakında eklenecekler: veritabanı (Supabase), fotoğraf yükleme, gerçek kullanıcı sistemi.
